# Rights and Licensing Boundary

This repository is a mixed-rights preservation project.

## MIT-covered material

The [LICENSE](LICENSE) file applies to Igor Lobanov-authored source code and authored documentation in this repository, except for the excluded paths listed below.

This MIT grant covers the repository's original Python implementation work, tests, scripts, and authored explanatory documentation unless a file is explicitly called out as excluded in this document.

## Excluded paths

The following tracked paths are intentionally excluded from the repository MIT grant:

- `src/alien_evolution/alienevolution/blocks.py`
- `figs/`
- `recordings/`
- `skool/`

These paths contain original-game-derived data, derivative preservation assets, or reverse-engineering artifacts tied to the original 1987 game. They are kept in the repository for preservation, research, documentation, and reproducibility purposes.

The original *Alien Evolution* game, including its title, audiovisual expression, gameplay assets, characters, and other original expressive elements, remains the property of its original authors and any other applicable rightsholders.

No rights in the original game IP are granted by this repository. In particular, the presence of excluded files or data in this repository does not grant permission to reuse, redistribute, commercialize, or relicence original-game-derived material beyond whatever rights you may otherwise have from the original rightsholders or applicable law.

## Project status and intent

This repository is an unaffiliated preservation and research project. It is not endorsed by, sponsored by, or formally affiliated with the original authors, publisher, or other rightsholders.

Any references in the repository to preservation, research, or non-commercial intent describe the purpose of the project. They do not add restrictions to the MIT license for files that are actually covered by the MIT license.
